# Assets Directory

Place your portfolio assets here:

## Required Files

1. **profile.jpg** - Your profile/headshot photo
   - Recommended size: 400x400px or higher (square)
   - Format: JPG, PNG, or WebP

2. **og-image.png** - Open Graph image for social sharing
   - Recommended size: 1200x630px
   - Format: PNG or JPG

## Optional Files

- Additional project screenshots
- Custom icons or graphics
